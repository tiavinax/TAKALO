# Takalo-takalo - Plateforme d'échange d'objets

## Description
Takalo-takalo est une plateforme web permettant aux utilisateurs d'échanger des objets entre eux. Les utilisateurs peuvent publier leurs objets, parcourir le catalogue des objets disponibles et proposer des échanges.

## Fonctionnalités
- ✅ Inscription et connexion des utilisateurs
- ✅ Gestion des objets (CRUD)
- ✅ Catalogue des objets disponibles
- ✅ Système de proposition d'échange
- ✅ Gestion des échanges (acceptation/refus)
- ✅ Interface responsive avec Bootstrap
